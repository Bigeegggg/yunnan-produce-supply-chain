# 品类内容策略手册

> 不同品类的产品，内容打法和消费者心理完全不同。本手册定义了五大品类的差异化内容策略。

---

## 水果类 🍊

### 消费者心理
- 购买驱动: 新鲜、甜度、性价比、产地信任
- 决策时间: 快（3-5秒决定是否感兴趣）
- 复购周期: 7-14天

### 最佳内容形式
| 形式 | 效果 | 说明 |
|---|---|---|
| 切开/榨汁展示 | ⭐⭐⭐⭐⭐ | 视觉冲击力最强，"想咬一口"的冲动 |
| 产地直拍 | ⭐⭐⭐⭐⭐ | 原生态 = 新鲜 = 信任 |
| 价格对比 | ⭐⭐⭐⭐ | "超市XX元，产地只要XX元" |
| 吃播测评 | ⭐⭐⭐⭐ | 真人食用增加信任 |
| 开箱体验 | ⭐⭐⭐ | 适合礼盒装水果 |

### 关键词策略
- 主词: 水果名 + "新鲜"/"当季"/"产地直发"
- 长尾词: "怎么挑选好的XX"、"XX什么时候最甜"

### 季节性日历
| 月份 | 应季水果 | 内容主题 |
|---|---|---|
| 1-2月 | 沃柑、脐橙 | 春节礼盒、冬日补充维C |
| 3-5月 | 枇杷、樱桃、蓝莓 | 春季尝鲜、亲子采摘 |
| 6-8月 | 芒果、葡萄、石榴 | 夏季消暑、冰镇吃法 |
| 9-10月 | 苹果、梨、柿子 | 秋季养生、润肺 |
| 11-12月 | 冰糖橙、柚子 | 年货预热、送礼 |

## 叶菜类/茄果类 🥬

### 消费者心理
- 购买驱动: 安全（农残焦虑）、新鲜度、产地环境
- 决策时间: 中等（10-15秒）
- 关键痛点: "市场上的菜到底打没打农药？"

### 最佳内容形式
| 形式 | 效果 | 说明 |
|---|---|---|
| 菜地实拍 | ⭐⭐⭐⭐⭐ | 看到种植环境 = 安心 |
| 从采摘到餐桌 | ⭐⭐⭐⭐⭐ | 全流程透明化 |
| 简单烹饪 | ⭐⭐⭐⭐ | 教会用户怎么做 |
| 农药检测展示 | ⭐⭐⭐⭐ | 直接打消顾虑 |
| 跟超市对比 | ⭐⭐⭐ | 新鲜度对比 |

### 关键词策略
- 主词: 菜品名 + "有机"/"绿色"/"新鲜采摘"
- 长尾词: "XX怎么做好吃"、"宝宝辅食蔬菜推荐"

### 信任建设要点
1. 务必展示种植环境（温室/大棚/露天菜地）
2. 如有检测报告，在视频中露出
3. 强调"从采摘到发货不超过X小时"

## 菌类 🍄

### 消费者心理
- 购买驱动: 稀缺性、野生/天然、营养价值、送礼体面
- 决策时间: 长（15-30秒，需要教育）
- 价格敏感度: 低（对高端菌类价格预期高）

### 最佳内容形式
| 形式 | 效果 | 说明 |
|---|---|---|
| 上山采菌过程 | ⭐⭐⭐⭐⭐ | "寻宝"式内容，观看完成率极高 |
| 生长环境揭秘 | ⭐⭐⭐⭐⭐ | 松茸为什么贵？看生长环境就懂了 |
| 品鉴/炖汤 | ⭐⭐⭐⭐ | 激发食欲 + 展示价值 |
| 真假辨别教学 | ⭐⭐⭐⭐ | 知识性内容，收藏率高 |
| 产量/稀缺性说明 | ⭐⭐⭐⭐ | 制造紧迫感 |

### 关键词策略
- 主词: 菌名 + "野生"/"云南"/"当季"
- 长尾词: "XX怎么辨别真假"、"XX和XX的区别"

### 注意事项
- 菌类属于**高风险品类**（涉及食用安全），务必：展示正规来源、标注"请勿自行采摘野生菌"等安全提示
- 高端菌类（松茸、松露、羊肚菌）走数字人/精品路线
- 日常菌类（香菇、平菇）走漫剧路线

## 根茎类 🥔

### 消费者心理
- 购买驱动: 品质、药食同源（三七/天麻）、家常必备（土豆/萝卜）
- 决策时间: 中等偏长
- **核心分化**: 日常根茎（走漫剧） vs 药材根茎（走数字人）

### 最佳内容形式
| 形式 | 效果 | 说明 |
|---|---|---|
| 挖采过程 | ⭐⭐⭐⭐⭐ | 类似菌类的"寻宝感" |
| 品质分辨教学 | ⭐⭐⭐⭐ | 教用户看外观、切面判断品质 |
| 炖汤/泡茶展示 | ⭐⭐⭐⭐ | 使用场景代入 |
| 产地文化故事 | ⭐⭐⭐⭐ | 文山三七、昭通天麻有文化底蕴 |
| 对比劣质品 | ⭐⭐⭐⭐ | 我们的vs市场上的，一眼看出差别 |

### 路线分流
| 产品 | 路线 | 原因 |
|---|---|---|
| 文山三七 | 精品/数字人 | 高价、药用、地理标志 |
| 昭通天麻 | 精品/数字人 | 高价、药用、送礼 |
| 土豆/萝卜/红薯 | 普通/漫剧 | 日常消费、低价 |
| 姜/蒜 | 普通/漫剧 | 调味品、日常消费 |

## 加工食品 🫙

### 消费者心理
- 购买驱动: 便利、健康、口味、品牌信赖
- 决策时间: 中等

### 最佳内容形式
| 形式 | 效果 | 说明 |
|---|---|---|
| 原料溯源 | ⭐⭐⭐⭐⭐ | 加工品的信任锚点在原料 |
| 加工过程透明化 | ⭐⭐⭐⭐⭐ | 干净、规范、无添加 |
| 多种吃法展示 | ⭐⭐⭐⭐ | 果干/果酱/花茶的N种用法 |
| 跟市售对比 | ⭐⭐⭐⭐ | 无添加 vs 有添加 |
| 包装拆箱 | ⭐⭐⭐ | 适合礼品属性产品 |

### 子品类策略
| 子品类 | 核心卖点 | 路线 |
|---|---|---|
| 果干/蜜饯 | 真材实料、无添加 | 漫剧 |
| 果酱/花酱 | 手工熬制、大果肉 | 漫剧 |
| 花茶/代用茶 | 原产地、养生 | 数字人 |
| 咖啡 | 云南小粒、精品 | 数字人 |
| 蜂蜜 | 原生态、纯天然 | 混合 |
| 速食/预制菜 | 方便、妈妈的味道 | 漫剧 |

## 跨品类通用原则

### 黄金前3秒法则
无论什么品类，前3秒必须做到以下之一：
1. **视觉冲击**: 切开/掰开/榨汁/蒸汽升腾
2. **价格冲击**: "超市卖XX，产地只要XX"
3. **认知反差**: "你以为的XX vs 实际上的XX"
4. **问题直击**: "你买的XX可能白吃了"

### 信任三件套
所有品类视频都应包含：
1. 👀 **产地实拍**（哪怕只有1秒）: 证明是源头
2. 🔬 **品质展示**（特写镜头）: 证明品质好
3. 💰 **价格对比**（可隐晦）: 证明性价比

### 内容复用矩阵

同一个产品的视频素材，可以按以下方式跨平台复用：

| 原始素材 | 抖音版 | 小红书版 | 视频号版 |
|---|---|---|---|
| 产地实拍 | 快剪15s + 热门BGM | 慢节奏45s + 详细讲解 | 故事化30s + 温情配乐 |
| 产品特写 | 钩子前3秒 | 封面图 | 中段信任建设 |
| 使用场景 | 中段欲望激发 | 全片种草 | 结尾社交传播 |

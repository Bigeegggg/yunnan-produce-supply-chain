# 品牌IP角色体系

> 可复用的漫剧角色库。同一品类复用同一角色，降低制作成本，建立品牌辨识度。

## 🧑‍🌾 宾川阿伯（水果类专用）

```yaml
角色档案:
  姓名: 阿伯老李
  年龄: 52岁
  外形: 圆脸、皮肤黝黑、戴着草帽、身穿蓝布衫
  性格: 热情豪爽、爱笑、说话带云南口音

  适用产品: 沃柑、葡萄、芒果、石榴、苹果等水果类
  
色彩方案:
  主色: "#F5A623" (橙黄 — 水果色)
  辅色: "#4A90D9" (蓝 — 天空/布衫)
  背景: "#FFF8E7" (暖白)

场景模板:
  - 果园全景: 高山背景 + 果树 + 蓝天
  - 采摘特写: 手部特写 + 果实 + 篮子
  - 切开展示: 水果剖面 + 果汁 + 惊讶表情
  - 打包发货: 纸箱 + 快递单 + 竖起大拇指
```

## 使用规则

### 复用原则
- 同一品类的不同产品（如宾川沃柑、元谋葡萄）共用同一角色
- 更换产品名+卖点文案即可，角色和场景模板不变
- 一个月更新一次角色服装/场景的小细节，保持新鲜感

### IP 扩展指南
- **不要过度设计**: 先跑通 1-2 个角色，验证效果后再扩展
- **角色不要太多**: 4 个角色覆盖所有品类足够，多了反而分散品牌认知
- **风格一致性**: 所有角色用同一画风（Q版+水彩），整体视觉统一

### 制作成本对照

| 方案 | 首次成本 | 复用成本 | 适用阶段 |
|---|---|---|---|
| 剪映图文成片 | 0元 | 5分钟/个 | MVP验证 |
| 来画动画 | ¥99/月 | 15分钟/个 | 批量化 |
| 外包插画师 | ¥200-500/角色 | ¥50-100/个 | 品牌化 |
| AI生成+人工精修 | ¥50/角色 | 10分钟/个 | 规模化 |

> **建议路径**: 先用剪映模板跑通 MVP → 验证有效后买来画会员批量化 → 数据稳定后外包统一角色设计
